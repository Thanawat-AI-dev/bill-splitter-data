เชื่อมต่อ GitHub

แนะนำให้ใช้ Firebase Firestore แผนฟรี สำหรับแชร์ให้เพื่อนหลายคน เพราะไม่ต้องฝัง GitHub token ในลิงก์ และรองรับการเขียนพร้อมกันได้ดีกว่า GitHub. GitHub ยังใช้ได้สำหรับเก็บไฟล์ส่วนตัว แต่จะไม่ใช้สร้างลิงก์ guest แล้ว

Firebase Firestore (แนะนำ / ฟรี)

วาง Firebase web config จาก Firebase Console แล้วกดบันทึก ใช้กับลิงก์แชร์ให้เพื่อน

GitHub mode เก็บโปรเจกต์เป็นไฟล์ JSON ในรีโพของคุณ โทเค็นจะอยู่ใน localStorage ของเบราว์เซอร์นี้เท่านั้น และไม่ถูกใส่ในลิงก์แชร์แล้ว
