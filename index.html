เชื่อมต่อ GitHub

ข้อมูลโปรเจกต์ทั้งหมดจะถูกบันทึกเป็นไฟล์ JSON ในรีโพ GitHub ของคุณ ผ่าน GitHub REST API โทเค็นจะถูกเก็บไว้ใน localStorage ของเบราว์เซอร์นี้เท่านั้น ไม่ถูกส่งไปที่อื่น แนะนำให้สร้าง Fine-grained Personal Access Token ที่จำกัดสิทธิ์เฉพาะรีโพนี้ และให้สิทธิ์ "Contents: Read and write" เท่านั้น
